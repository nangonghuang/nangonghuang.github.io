  2. windows下，编写一个bat : start venv\scripts\activate.bat ，双击bat启动虚拟环境

    linux下，terminal中执行 source venv/bin/activate 启动虚拟环境

  3. 进入虚拟环境，pip install flask

  4. 新建python文件，输入

  5. 在vscode中ctrl+p, 输入>python ,选择调试器，选择venv,这时候flask应该不会报错了

  6. 虚拟环境中直接运行python文件
